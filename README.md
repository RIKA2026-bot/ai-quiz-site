מדריך כלי AI לשאלונים 🚀

אתר אינטראקטיבי ומעוצב המרכז את כלי הבינה המלאכותית המובילים ליצירת שאלונים, חידונים וקוויזים.
האתר נבנה במטרה להקל על מורים, מרצים ומדריכים לשלב למידה אינטראקטיבית, ומכיל סרטוני הדרכה, אפשרויות להורדת קובצי PDF, ומצגות מוטמעות (Canva ו-YouTube) בתוך האתר עצמו.

🛠️ כלים נסקרים באתר:

Quizlet: יצירת כרטיסיות ושאלות מתוך טקסט לצורך תרגול ולמידה עצמית.

MagicSchool AI: כלי ייעודי למורים שמייצר שאלות, קוויזים ודפי עבודה.

QuestionWell: כלי שמייצר שאלות אמריקאיות ומאפשר ייצוא ישיר ל־Google Forms.

Edpuzzle: הוספת שאלות אינטראקטיביות בתוך סרטוני וידאו.

Edcafe.ai: יצירת יחידות מיקרו-למידה, שאלונים חכמים ומסלולי למידה.

💻 טכנולוגיות ומאפיינים

האתר בנוי כעמוד אינטרנט בודד (Single Page) מהיר ורספונסיבי:

HTML5 ו-CSS3 לעיצוב נקי וכהה (Dark Mode).

JavaScript (Vanilla) להפעלת חלונות קופצים (Modals) דינמיים למצגות.

תפריט ניווט דביק (Sticky Nav) כולל לוגו, קישורים מהירים, וכפתור חכם.

עיצוב רספונסיבי המותאם אישית למחשב, טאבלט וטלפון נייד.

🌍 איך מפרסמים את האתר ב-GitHub Pages?

צרו מאגר (Repository) חדש ב-GitHub.

העלו למאגר את כל הקבצים הנדרשים שפורטו מעלה.

ב-GitHub, היכנסו ללשונית ההגדרות (Settings) -> ואז לתפריט צד (Pages).

תחת Source / Branch בחרו בענף main (או master) ולחצו Save.
